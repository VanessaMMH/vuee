# Microfront-end erp-mf-punto-venta

## Comando para instalar dependencias
```
npm install
```

### Comando para compilar y levantar el proyecto en modo desarrollo
```
npm run serve
```

### Comando para generar los archivos en modo produccion
```
npm run build
```
